TUTORIALS


MODEL-BUILDING & SIMULATION TUTORIALS

The Model-building and Simulation Tutorials will acquaint you with how to use the software to construct computer-simulation models.

Tutorial 1: Stocks & Flows; Graphs & Tables

Tutorial 2: Converters & Connectors; Graphical Functions; WYSIWYG algebra; Ghosts

Tutorial 3: Decision Process Diamonds

Tutorial 4: Sensitivity Analysis

Tutorial 5: Sector Frames; Run Specs; Trace Logic from a Graph


INTERFACE CREATION TUTORIALS

The Interface Creation Tutorials will help you to understand how to use the interface-creation features of the software to share your insights and understanding with others.

Tutorial 1: Text Block; Info Buttons; Menu Control Buttons; Graphics Frame

Tutorial 2: Story Telling

Tutorial 3: Navigation Buttons; Switches; Numeric Displays

Tutorial 4: Warning Devices (Flashers); Knobs; Sliders; Graphical Input Devices


COMPLETED TUTORIAL MODEL

When you complete all of the tutorials, you will have a running STELLA model, complete with a user-interface. An example of a completed tutorial model has been provided in the Tutorials folder.
